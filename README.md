# Investment_analysis

## Analysis of stocks FAANG

### Run locally
Install dependencies in a virtual environment is recommended.

### Virtual environment and Libraries in Linux/Windows:

```python 
Linux:
  python3 -m venv env
  source ./env/bin/activate
  pip3 install -r requirements.txt
  deactivate
  
Windows:
  python3 -m venv env
  .\env\Scripts\activate
  pip3 install -r requirements.txt
  deactivate
```

 ### API Parameter Descriptions

### **api_key**
* Got to https://www.alphavantage.co/, to get your api_key. 
* Then do not forget to set your environment variable AV_API_KEY to API key.
